HTML to Image

`textarea` に HTML を貼り付けて、ブラウザだけで `canvas` 描画と PNG 化まで行う単一ページです。 外部ライブラリは使わず、GitHub Pages にそのまま配置できます。

  • 入力 HTML はインライン CSS 前提です。
  • 外部画像や外部フォントは CORS 制約で失敗する場合があります。
  • Safari など一部環境では `foreignObject` の制約で描画差異が出ます。GitHub Pages 上では Chrome 系ブラウザ想定が安定です。
