foreignObject 差異テスト用 HTML

下の 3 パターンは、メインの変換ページの textarea にそのまま貼れる生 HTML です。 ゲームのお知らせ画像の安定パターン、差異確認用の演出強めパターン、ニュース記事っぽい本文レイアウトを入れています。

Pattern 1: Game Notice / Stable

ふつうのゲーム告知画像を想定した安定寄りのパターンです。大きな見出し、更新内容、開催期間、CTA を入れつつ、比較的ズレにくい CSS に寄せています。

Pattern 2: Game Notice / Stress

同じくゲーム告知テーマですが、backdrop-filter、mix-blend-mode、絵文字、絶対配置、回転、強い字詰めを入れてあります。差異確認用です。

Pattern 3: News Article

記事本文っぽいレイアウトです。見出し、公開日、リード文、小見出し、本文、箇条書きを含むので、ニュースやお知らせ本文の確認に使えます。
